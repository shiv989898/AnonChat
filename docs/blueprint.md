# **App Name**: AnonChat

## Core Features:

- Anonymous Matchmaking: Connect users randomly for 1-on-1 text chat.
- Real-Time Chat: Enable real-time text exchange between matched users using Socket.io.
- Next Partner: Allow users to disconnect and find a new random partner.
- Message Timestamps: Display timestamps for each message sent.
- Profanity Filter: Implement a tool to filter out basic profanity in messages.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to create a calming and neutral atmosphere.
- Background color: Light gray (#F5F5F5), a desaturated shade of lavender, for a clean and modern look.
- Accent color: Pale blue (#B0E2FF) to highlight interactive elements like buttons and links, offering a subtle contrast.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and neutral look.
- Simple, minimalist icons for actions like 'next' and 'leave'.
- Mobile-responsive layout for accessibility on various devices.
- Subtle animations for connection and disconnection events.
